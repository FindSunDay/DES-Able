This ‘original_dataset’ folder contains the following items, which are unprocessed datasets from the original resources.


1. DES_monthly_reports  
      
Subdirectory contains 111 DES performance month reports in excel and pdf formats. They are the input for ‘des_performance.py’.

2. DES contact list.xlsx

An excel file contains DES provider information including name, website, email and site location. These information were collected manually from online resources. Refer to the Support Document for the details. It is the input file for ‘des_data.py’.

3. des-star-ratings-march-2020.xlsx

An excel file stores a list of DES providers in Australia, which was collected from the Department of Social Services. It is the input file for ‘des_data.py’.
